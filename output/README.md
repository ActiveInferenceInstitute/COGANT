# COGANT Output Directory

This directory functions as the destination for compiled artifacts and exports generated by the COGANT build pipeline.

## Structure

*   `pdf/`: For combined PDF manuscript rendering (e.g., `cogant_combined.pdf`) once the project is fully integrated.
*   `slides/`: Optional presentation slides generated alongside the manuscript.
*   `web/`: Optional HTML outputs for the project.

## Note on Pipeline Execution

Since this project resides within `projects_in_progress`, it is isolated from the automated PDF-rendering routines of `infrastructure/` tools. Once the project is promoted to the active `projects/` directory, pipeline scripts like `scripts/03_render_pdf.py --project cogant` will output generated figures, cross-reference data, and final PDF artifacts directly into this folder.
