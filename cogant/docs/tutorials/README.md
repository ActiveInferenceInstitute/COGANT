> Step-by-step, hands-on walkthroughs that take you from a blank shell to a fully analyzed, custom-ruled, plugin-extended COGANT workflow. Each tutorial is self-contained and runnable; later tutorials build on the vocabulary introduced earlier.

## Recommended Reading Order

1. [01 Quickstart](01_quickstart.md) — get a working install and your first scan in under five minutes.
2. [02 Small Repo Walkthrough](02_small_repo_walkthrough.md) — tour every stage of the pipeline on a controlled input.
3. [03 Flask Walkthrough](03_flask_walkthrough.md) — see how COGANT behaves on a non-trivial real codebase.
4. [05 GNN Interpretation](05_gnn_interpretation.md) — learn to read the artifacts you just generated.
5. [04 Custom Rules](04_custom_rules.md) — start customizing the translation engine.
6. [06 Reverse Mode](06_reverse_mode.md) — close the roundtrip by going from GNN back to code.
7. [07 Plugin Authoring](07_plugin_authoring.md) — extend COGANT with your own components.
