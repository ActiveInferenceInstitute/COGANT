## API Stability

The COGANT Python API is currently in `0.6.0` (pre-1.0). Breaking changes may occur on minor-version bumps until the `1.0.0` release. For production use, pin your dependency version:

```
cogant==0.6.0
```

See the site [changelog](../changelog.md) for per-release breaking-change notes. Repository-level release-cadence and deprecation policy notes live in `CONTRIBUTING.md#release-cadence`.
