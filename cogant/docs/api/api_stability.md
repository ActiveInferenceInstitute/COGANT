## API Stability

The COGANT Python API is currently in `0.5.0` (pre-1.0). Breaking changes may occur on minor-version bumps until the `1.0.0` release. For production use, pin your dependency version:

```
cogant==0.5.0
```

See [`CHANGELOG.md`](../../CHANGELOG.md) for the per-release breaking-change notes and [`CONTRIBUTING.md`](../../CONTRIBUTING.md#release-cadence) for the release cadence and the deprecation policy that applies between versions.
