## Security Release Process

### Vulnerability Discovery

If you discover a vulnerability:

1. **Report confidentially**: security@cogant.io (future)
2. **Do not disclose**: Publicly avoid until fixed
3. **Include**:
   - Description
   - Affected versions
   - Reproduction steps
   - Proof of concept (optional)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Triage**: Within 1 week
- **Fix**: As soon as possible (prioritized)
- **Release**: Security patch release
- **Disclosure**: Public announcement after release

### Support Lifecycle

| Version | Release | End of Life |
|---------|---------|-------------|
| 1.x | Q4 2025 | Q4 2026 |
| 0.x | < Q1 2025 | Q4 2025 |

Security patches provided for latest version + previous major.
