## Responsible Disclosure

Security researchers are encouraged to report vulnerabilities privately before public disclosure. Use the contact or reporting path given in [RFC 0001](https://github.com/cogant-contributors/cogant/blob/main/cogant/specs/rfc/0001-name-and-scope.md); if none is listed there, use the repository **Security** advisories flow or reach maintainers through a non-public channel they publish for this project.

Operational validation of analysis outputs (bundle checks, GNN package scoring) is documented under [VALIDATION](../validation/README.md) and the [`cogant validate`](../cli/README.md) command.
