# COGANT CLI Guide

Related: [Documentation hub](./README.md) · [Python API](../api/README.md) · [Architecture](../architecture/README.md)

The COGANT Command-Line Interface provides tools for analyzing codebases and translating them into Generalized Notation Notation (Active Inference Institute) state-space and process-model representations. Note: "GNN" in COGANT refers to Generalized Notation Notation, not graph neural networks.
