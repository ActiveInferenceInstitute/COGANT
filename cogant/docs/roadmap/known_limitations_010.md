## Known Limitations (0.1.0)

1. **Single-language**: Python only (v0.1)
2. **No dynamic analysis**: Static analysis only
3. **No cross-codebase**: Single project at a time
4. **Limited type inference**: Relies on annotations
5. **No IDE integration**: CLI/Python API only
6. **Visualization**: Basic HTML site via `cogant render`; advanced graph UX is roadmap
