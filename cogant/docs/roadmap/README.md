# Roadmap

> Where COGANT is going and how it plans to get there. Version plans,
> feature backlog, performance and coverage targets, deprecation policy,
> community milestones, and the changelog are all linked from here.
> Read this section to understand the release strategy, to check whether
> your favourite feature is on the list, or to see what breaking changes
> are coming.
