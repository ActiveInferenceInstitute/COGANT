#### 3.4 ReviewManager (`cogant/translate/review.py`)

**Purpose:** Manage human review and curation of semantic mappings.

**Review Actions:**
- `accept_mapping()` - Approve as valid (upgrades tier)
- `reject_mapping()` - Mark as invalid
- `edit_mapping()` - Modify based on feedback
- `split_mapping()` - Decompose into multiple mappings
- `merge_mappings()` - Combine multiple mappings

**Key Methods:**
- `add_mapping()` - Add for review
- `get_unreviewed_mappings()` - All auto_proposed
- `get_mappings_by_status()` - Filter by status
- `get_review_summary()` - Statistics
- `export_reviewed_mappings()` - Get curated results
- `get_review_history()` - Audit trail

**Status Values:**
- `auto_proposed` - Generated by rules
- `accepted` - Human approved
- `rejected` - Human rejected
- `edited` - Modified by human
- `split` - Decomposed into multiple
- `merged` - Combined with others

from cogant.translate.review import ReviewManager
