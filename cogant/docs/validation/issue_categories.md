## Issue Categories

| Category | Typical Issues | Level |
|----------|---|---|
| Coverage | Uncovered files, skipped functions | WARNING |
| Confidence | Low confidence nodes/edges | INFO/WARNING |
| Consistency | Orphaned nodes, cycles | WARNING |
| Schema | Invalid field values | ERROR/FATAL |
| Reproducibility | Hash mismatch | WARNING |
| Performance | Excessive memory, timeout | ERROR/FATAL |
